# 🏫 《可堡灵之校》游戏设计大纲

> 版本：v0.1-alpha | 引擎：Ren'Py | 类型：校园悬疑·生存·文字AVG

---

## 一、核心概念

> **「一所被封锁的学校。一个变成怪物的同学。一周之内，找到真相。」**

一场空间的异变将整所学校与外界隔绝。昔日同窗「可堡灵」异化为校园的统治者，强制推行以「可丽币」为核心的生存法则。玩家需要在七天之内探索校园、收集线索、累积信任，最终揭开「堡灵之印」的秘密，决定自己的命运。

---

## 二、世界观设定

### 2.1 背景
- **地点**：一所普通的寄宿制中学
- **时间**：现代，一周目固定为7天
- **异变**：某天深夜3:44，空间发生「重叠」，学校被半透明雾气封锁，所有电子设备失效

### 2.2 核心谜团
- **堡灵之印**：封锁的核心，即可堡灵力量的来源。外观为一枚吊坠，摧毁即可解除封锁
- **时空重叠**：学校100年前发生过类似事件，历史正在重演
- **记忆消除**：离开雾气的人会被抹去一切记忆，包括对外界的认知

### 2.3 规则体系
| 规则 | 内容 |
|------|------|
| 第一条 | 晚上十点后禁止离开宿舍 |
| 第二条 | 禁止靠近学校大门 |
| 第三条 | 一切以可丽币结算 |

## 三、角色设定

### 🟥 可堡灵 (KBL)
| 属性 | 内容 |
|------|------|
| 绰号 | 「堡灵哥」 |
| 外表 | 黑发，校服笔挺，眼睛在暗处发红光 |
| 常态 | 吊儿郎当、不写作业 |
| 异变后 | 冷酷、掌控欲强、说一不二 |
| 定位 | **核心反派**，也是解谜的关键 |
| 信任值 | `kbf_trust`（影响结局分支） |

### 🟦 林晓 (LX)
| 属性 | 内容 |
|------|------|
| 身份 | 前学生会成员 |
| 性格 | 谨慎、细心、善良 |
| 隐藏技能 | 持有所有楼的钥匙 |
| 据点 | 旧钟楼顶层（第五天后） |
| 定位 | **盟友/攻略角色** |
| 信任值 | `lx_trust`（影响信息获取） |

### 玩家（主角）
- **姓名**：玩家自定义（最多12字）
- **默认名**：master
- **视角**：第一人称
- **属性**：精力、可丽币、信任值

## 四、游戏系统

### 4.1 基础属性

| 属性 | 范围 | 说明 |
|------|------|------|
| 体力/精力 | 0~100 | 行动消耗，每日结束自动恢复15点。归零→Game Over |
| 可丽币 | 0+ | 校内货币，用于购买食物、信息等 |
| KBL信任 | -∞~∞ | 影响可堡灵对你的态度和结局分支 |
| LX信任 | -∞~∞ | 影响林晓提供信息的多寡 |

### 4.2 资源获取
- **可丽币**：探索拾取、完成支线、交易
- **食物**：食堂购买（3币/份，+20体力）、探索拾取
- **据点**：三楼储物间（初期）、钟楼（后期）
- **纸条/笔记**：线索载体，推动剧情

### 4.3 日程结构
```
每天 = [早晨 → 白天探索（含选择肢）→ 傍晚 → 夜晚事件]
```
- 每天的剧情推进由 `day_end_check()` 控制
- 每周目固定7天
- 体力不足时触发 Game Over

## 五、剧情结构 — 一周目完整流程

### 第1天 · 平凡的世界
- 普通周一早晨 → 可堡灵没写作业 → 上课铃 → 可堡灵被老师罚站时「变成另一个人」
- 午饭与林晓对话 → 傍晚可堡灵逃课 → 夜间3:44苏醒 → 雾气封锁校园
- **关键事件**：广播宣布三条规则

### 第2天 · 宿舍探索
- 发现纸条 → 三楼储物间（获得可丽币/铁盒线索）或搜索宿舍（获得食物）
- 食堂规则 → 物价昂贵，第一次生存抉择
- **关键道具**：旧学生证 / 铁盒

### 第3天 · 校园探索
- 储物间新纸条：「第五天晚上，钟楼见」
- 食堂再遇林晓 → 得知后山器材室 → 走廊遭遇可堡灵（红光眼）
- **关键事件**：确认可堡灵已非人类

### 第4天 · 夜晚异响
- 校园人口锐减 → 深夜敲门声
- 开门获得物资和鼓励纸条 / 不开门避免暴露
- **氛围**：孤独与恐惧加剧

### 第5天 · 钟楼之约
- 前往旧钟楼 → 见到林晓的秘密据点
- 得知「堡灵之印」情报 → 百年历史的轮回
- **目标确立**：找到并摧毁堡灵之印

### 第6天 · 绝境
- 图书馆/行政楼均被封锁 → 巡逻队增多
- 集会告示：「明天下午五点·操场」
- 可堡灵手下抓人 → 选择跟随或逃跑
- **终局逼近**

### 第7天 · 终章
- 全校集会 → 可堡灵宣布新制度
- **三选一结局**：
  1. 🚪 **离开**（遗忘结局）— 走出校门，失去一切记忆
  2. ⚔️ **抗争**（苏醒结局）— 夺走堡灵之印，解除封锁
  3. 👁 **观察**（潜伏结局）— 留在学校，等待二周目
