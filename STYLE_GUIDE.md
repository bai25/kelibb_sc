# 《可堡灵之校》画风参考

## 参考图
`game/images/style_ref.jpg`

## ✅ 要的
- 吉卜力（Ghibli）式暖色调：金色、奶油色、大地色系
- 新海诚（Shinkai）式光影：柔和的自然光、漫反射、暖黄光源
- 干净日系校园风
- 复古木质建筑元素（旧校舍感）
- 角色暖皮、柔和线条

## ❌ 不要的
- 蒸汽朋克 / 齿轮 / 钟表机械
- 未来科技 / 赛博朋克
- 哥特暗黑
- 冷白皮 / 荧光色
- 过度写实

## 角色立绘要求
- 半身、白底、透明PNG
- 色彩温暖柔和
- 线条干净，不过分锐利
- 不要机械装饰

## 场景要求
- 校园日常场景
- 暖色调自然光
- 干净整洁
