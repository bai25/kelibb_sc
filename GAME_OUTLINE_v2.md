## 一、核心概念

> **「六个人格。六名学生代表。一场不存在的春游。」**

校长被神秘的面具人取代，后者拥有六重人格，需要「重塑」六个学生的人格来完善自己。六名学生代表被选入这场阴谋，其中五人身怀异能，唯独主角——只是一个普通人。

---

## 二、世界观与设定

### 2.1 基础信息
- **主角**：吕文强（谐音名，使用原林晓立绘）
- **学校**：不变
- **核心反派**：面具校长（六重人格分裂）
- **商人**：可丽堡（原名可堡灵，名字微调，保留可丽币系统）

### 2.2 规则体系
- **w 周目代号**：w1 = 一周目，w1d1 = 一周目第一天
- **货币**：可丽币（保留）

- **角色死亡**：主角阵亡后切换至剩余5名学生代表之一继续

## 三、剧情流程

### w1d1 · 异常的前兆

**早晨 ~ 下午**
- 主角正常学习
- [环境细节] 总发现自己的课本被动过，杯子被动过
- 怀疑自己记错，但隐约觉得不对

**下午一点**
- 班主任宣布春游一事
- 同学们欢呼，主角也跟着欢呼
- [氛围] 春游的消息冲淡了主角的不安

**晚餐**
- 同学们都在讨论春游
- 主角不小心听到：本次春游校长会选举6名学生代表
- [悬念] 学生代表？春游为什么需要选举代表？

**夜晚**
- 主角学习一天，疲惫入睡
- → **w1d1 结束**

### w1d2 · 学生代表

**早晨**
- 室友们都起得很早，兴奋地讨论春游
- 主角来到教室正常上早自习

**上午**
- 班主任来到教室，叫吕文强出去
- 宣布他是六名学生代表之一
- 要求在大集合之前前往校长办公室
- [选择肢] 主角的反应

**【分支剧情 · 校长办公室】**
- 主角与其他5名学生代表来到校长办公室
- 校长迟迟不出现
- 主角怀疑校长食言，决定出去看看

**【发现异常】**
- 集合地点空无一人！
- 全校师生消失了

**【选择肢 A：尝试离开校园】**
- → 主角走向校门
- → 被黑衣人拦住，不允许离开
- → 主角强行冲突
- → 发生争执，被击晕
- → **主角阵亡** → 进入 **w2**
- → 选择剩余5人之一继续游戏

**【选择肢 B：前往教室办公室获取手机】**
- → 办公室上锁
- → [尝试打开] → 成功打开
- → （系统提示彩蛋：这是在犯罪）
- → 手机没有信号
- → 跳转至广播事件

**【选择肢 C：离开】**
- → 跳转至广播事件

**广播事件**
- 广播响起，宣布延期本次春游
- 学生们都回来了
- 但主角发现有点不对劲……
- **同学们格外的认真……**
- [恐怖氛围] 过分安静、过分专注、眼神空洞

## 四、核心谜团设定

### 4.1 面具校长
- **身份**：取代了原校长
- **本质**：六重人格分裂者
- **目的**：重塑六名学生的人格，完善自身
- **能力**：控制学校空间、操控师生

### 4.2 六人格设定（待细化）

|----------|----------|----------|------|
| 1 | 待定 | 待定 | - |
| 2 | 待定 | 待定 | - |
| 3 | 待定 | 待定 | - |
| 4 | 待定 | 待定 | - |
| 5 | 待定 | 待定 | - |
| 6 | 待定 | 待定 | 最终人格？ |

### 4.3 普通人的意义
- 主角（吕文强）是唯一没有异能的人
- 暗示：普通本身可能是一种「能力」——对精神控制的免疫？
- 作为玩家视角的锚点，让玩家与主角共情

## 五、系统设计

### 5.1 精力系统（改造版）
| 项目 | 数值 |
|------|------|
| 上限 | 50（初始） |
| 恢复 | 每日自动恢复 |
| 提升 | 通过道具/事件提升上限 |
| 归零 | 行动受限/剧情分支 |

### 5.2 可丽币
- 商人：可丽堡（原可堡灵改版）
- 用途：购买道具、情报、能力提升

### 5.3 角色切换系统
- 主角阵亡后切换至剩余学生代表
- 不同角色拥有不同能力和视角
- 多角色多周目解锁完整故事

### 5.4 分支/多周目
- w1 = 一周目（主角视角）
- w2 = 主角阵亡后的替代角色视角
- 多周目解锁真相
